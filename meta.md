```// Given a binary tree of integer nodes, find the subtree with the maximum sum, and return this sum.
// Input :
//      2

Given a sequence of integers and an integer total target, 
return whether a continuous sequence of integers sums up to target.

Example
[1, 3, 1, 4, 23], 8 => True (because 3 + 1 + 4 = 8)
[1, 3, 1, 4, 23], 7 => False
```
